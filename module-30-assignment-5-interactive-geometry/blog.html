Q1 : What is DOM and what is the purpose of DOM?


Answer: DOM is a standard programming interface that models the HTML and XML document elements as a tree of objects. It allows us to access, add, edit, remove any of the HTML or XML documents data as we need and make the page interactive with Javascript in a platform neutral way.


Q2 : How will you select HTML elements using DOM? Name the DOM methods.


Answer: We can use 5 DOM methods to select HTML elements using DOM. 1. document.getElementById(id name) 2. document.getElementsByTagName(tag name) 3. document.getElementsByClassName(class name) 4. document.querySelector(selector) 5. document.querySelectorAll(selector)


Q3 : What is event bubble?


Answer: Event bubble is the process when an event triggers on an element, it travels upwards and triggers the same event in the parent Nodes of the DOM tree until it reaches the root.


Q4 : What is a callback function and why will you use it?


Answer: Callback function is a function that is passed as a parameter to another function. When we want to call the callback function after some tasks in the function is done, we can use the callback function.
